Modelo de Ruteo con Entregas Fraccionadas

Optimización avanzada usando OR-Tools

Desarrollo e implementación de un algoritmo de optimización para el problema de ruteo de vehículos con entregas fraccionadas (Split Delivery Vehicle Routing Problem). El proyecto logró reducir costos operativos en un 25% y mejorar la eficiencia de las rutas de distribución.

Python OR-Tools Google Cloud REST API
25% Reducción de costos
40% Menos tiempo de cómputo
15+ Rutas optimizadas diariamente
